# Frontend Neovim Config

A minimal, efficient, opinionated (currently) neovim configuration

## Features

- Space as `<leader>`
- Typescript, React, Node, Lua, Astro, Vue, TailwindCSS
- Lazy plugin loading
- LSP
- AI support with Supermaven and Opencode
- Formatting and linting with Conform/nvim-lint with support for oxlint/oxfmt,
  biome, eslint, prettier, and stylua
- Lazygit
- Tmux navigation
- Snacks.nvim

## Install

```sh
git clone git@github.com:loremipson/nvim.git ~/.config
```

### Prerequisites

- Neovim 0.11+
