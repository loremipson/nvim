# Frontend Neovim Config

A minimal, efficient, opinionated (currently) neovim configuration

## Features

- Space as `<leader>`
- Typescript, React, Next, Node, Astro, Vue, TailwindCSS
- Lazy plugin loading
- LSP
- AI support with Supermaven
- Formatting and linting with Conform/nvim-lint
- Lazygit
- Tmux navigation
- Snacks.nvim

## Install

```sh
git clone git@github.com:loremipson/nvim.git ~/.config
```

### Prerequisites

- Neovim 0.11+
